# Регулярные выражения
`finditer(паттерн, строка)` - все вхождения строк
`x.group()` - конкретная строка вхождения
Использовать нужно неэкранированные строки: `r"[a-z]+\.[a-z]+"`

`A*` - пустота или буква А
`А+` - Буква А (от 1 шт. до бесконечности)
`A{3}` - ААА (ровно 3 буквы А)
`A{1,3}` - A, АА, ААА (от одной до трёх букв)

`[ABC]` - один из символов A, B или C
`[ABC]{4}` - BCAA, CABB
`[A-Z]` - один из символов от A до Z
`[0-9]{3}` - 132, 607
`[^CDE]` - не C, не D, не E

`.` - любой произвольный символ

`(AB)+` - пары AB идут подряд
`(?=(...))` - все возможные вхождения с учётом пересечения комбинаций. Желательно всего использовать этот приём. С ним нужно писать `x.group(1)`
`AB|CD|DE` - AB или CD или DE. Сочетания нужно записывать в порядке убывания их длины: `YZZ|XY`

**ВАЖНО**: 
1. если служебные символы проверятся как обычные, то перед ними надо поставить \ (`\. \+`)
2. если одну регулярку нужно будет вставлять в другую с помощью f-строк, первую регулярку нужно оборачивать в скобки: `num = r'([1-9]*)'`
### 1
Текстовый файл 24-239.txt состоит не более чем из 106 символов и содержит только заглавные буквы латинского алфавита Х, У и Z. Найдите максимальную длину подстроки, которая состоит из сочетаний XY, YZ, YZZ, записанных в произвольном порядке. В ответе укажите наибольшую длину подходящей подстроки. Например, в строке ZZXZXZZXYYZYZZYYY самая длинная подходящая подстрока ХҮYZYZZ имеет длину
```python
from re import *
s = open('ege2026kp/24data/24-239.txt').readline()
reg = r'(YZZ|XY|YZ)+'
reg = rf'(?=({reg}))'
m = max([x.group(1) for x in finditer(reg, s)], key=len)
print(len(m), m)
```

### 6
Текстовый файл состоит не более, чем из 10^7 символов из набора A, B, C, D, E, F.
Найдите максимальную длину строки вида `А*А*А*А`, где между буквами А расположены одинаковые группы символов, не содержащие букв А. Например, в строке BDADBADBADBABDAFABDA такая подстрока ADBADBADBA (длина 10).
```python
from re import *
s = open('24').readline()
reg = r'A[B-F]+A[B-F]+A[B-F]+A'
reg = rf'(?=({reg}))'
m = 0
for x in finditer(reg, s):
    a = x.group(1).split('A')
    if a[1] == a[2] == a[3]:
        m = max(m, len(x.group(1)))
print(m)
```
