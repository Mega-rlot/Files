# Команды

| Команда исполнителя в задании | Команда исполнителя в коде     | Описание команды                                                                                                                                                              |
| ----------------------------- | ------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Опустить хвост                | pendown() pd() down()          | Опускает хвост черепахи. Теперь при движении исполнителя за ним остается след в виде линии                                                                                    |
| Поднять хвост                 | penup() pu() up()              | Поднимает хвост. Теперь при движении исполнителя след оставаться не будет                                                                                                     |
| Вперед х                      | forward(x) fd(x)               | Перемещает исполнитель на х вперед. При отрицательном значении х перемещает исполнителя назад с сохранением ориентации (без поворота на 180 градусов). Вперед (-х) = Назад(х) |
| Назад х                       | backward(x) bk(x) back(x)      | Перемещает исполнитель на х назад. При отрицательном значении х перемещает исполнителя вперед на х (без поворота на 180 градусов). Назад(-х) = Вперед(х)                      |
| Направо х                     | right(x) rt(x)                 | Поворот исполнителя на х градусов по часовой стрелке. При отрицательном значении х поворачивает исполнителя против часовой стрелки. Вправо(-х) = Влево(х)                     |
| Налево х                      | left(x) It(x)                  | Поворот исполнителя на х градусов против часовой стрелки. При отрицательном значении х поворачивает исполнителя по часовой стрелке. Влево(-х) = Вправо(х)                     |
| Повторить х раз [Команды]     | for __ in range(x):<br>команды | Повторяет все команды, записанные с отступом от начала строки, x раз                                                                                                          |

# Пример 1:
```python
from turtle import *

tracer(0)  # Выключаем анимацию

r = 15
lt(90)
for __ in range(70):
    fd(8 * r)
    rt(30)

# Нанесение точек (обязательно)
pu()

for x in range(-20, 20):
    for y in range(-20, 20):
        goto(x * r, y * r)
        dot(3)

update()  # Обновляем экран после всех операций
done()
```
