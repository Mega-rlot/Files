# Способ с прямоугольниками
- Выбираем "Вставка", "Фигура", "Базовые фигуры", "Прямоугольник", подписываем номер на каждом из них и расставляем прямоугольники, учитывая зависимости и длительность процессов
# Ручной способ
Определения минимального времени завершения выполнения всех процессов:
1. Создаём столбец "Реальное время"
2. Напротив независимых процессов в этом столбце дублируем время выполнения (например, =B2)
3. Для зависимых процессов мы берём "реальное время" процесса, от которого он зависит, и прибавляем время выполнения (=D14+B4)
4. Для процессов, зависимых от нескольких процессов, берём МАКС от "реального времени" этих процессов и прибавляем время выполнения (=МАКС(D6;D8)+B7)
5. Берём МАКС от всех ячеек столбца (реальное время)
# Автоматический способ
1. Разбиваем текст по столбцам: данные - текст по столбцам - в параметрах разделения точка с запятой
2. Вместо заголовков столбцов добавляем нулевой процесс с временем выполнения
3. Выделяем столбцы с зависимостями, открываем "Найти и заменить", заменяем пустые значения на 0 и "Заменить всё"
4. После всех столбцов напротив процесса в строке 2 пишем формулу с ВПР, где указываем:
	1. всю таблицу, включая текущую ячейку, далее пишем номер текущего столбца и 0. 
	2. Если хоть какой-либо из процессов имеет несколько "родительских" процессов, то для каждого процесса пишем столько ВПРов, на сколько столбцов мы разделили тот столбец в прошлом шаге (в этом примере 2 столбца, но может быть больше). 
	3. В конце прибавляем время выполнения этого процесса: $$=МАКС(ВПР(C2;A:E;5;0);(ВПР(D2;A:E;5;0)+B2$$
5. Растягиваем формулу с ВПР вниз
6. Далее действуем в зависимости от условия:
	1. Чтобы определить минимальное время выполнения всех процессов, то берём максимальное значение из последнего столбца
