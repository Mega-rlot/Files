### Перевод чисел из десятичной системы счисления
Перевод в двоичную систему: `bin(number)`
Перевод в восьмеричную систему: `oct(number)`
Перевод в шестнадцатеричную систему: `hex(number)`

Если не нужен префикс результата перевода, то нужно взять срез у полученной строки: `bin(number)[2:]`

Так же выводить числа в других системах счисления можно используя f-строки и формат вывода. Для этого в строке, через символ : указываем буквы **b** — для двоичной, **o** — для восьмеричной и **x** / **X** — для шестнадцатеричной системы счисления.
```python
print(f'Двоичное: {number:b}')
```

### Синтаксис f-строк
print(f'текст {переменная} текст')

# Пример 1
Сколько единиц содержится в двоичной записи значения выражения: 4^2020 + 2^2017 - 15?
**Решение:**
Преобразуем выражение:
4^2020+2^2017-15 = (2^2020)^2 + 2^2017-15 = 2^4040 +2^2017-2^4+2^0.
Число 2^4040 в двоичной записи записывается как единица и 4040 нулей. Добавив число 2^2017, получаем 100...00100... (единица, 2022 нулей, единица, 2017 нулей, всего 4041 разрядных цифр). Если вычесть из этого числа 2^4 = 10000 прибавить 2^0, то число примет вид 100...001...10001. В полученном числе единица, 2023 нуля, 2013 единиц, три нуля и о единица. Значит, всего в числе 2015 единиц.
Ответ: 2015
**Код:**
```python
from string import printable

n = 4**2020 + 2**2017 - 15
s = ''
while n:
    s = printable[n % 2] + s
    n //= 2
print(s.count('1'))
```
**Код:**
```python
print(bin(4**2020+2**2017-15)[2:].count('1'))
```

# Пример 8
При каком наименьшем натуральном значении переменной x в выражении $36^{17}-6^{x}+71$ сумма цифр в шестеричной записи числа равна 61?
```python
# функция для подсчёта суммы цифр
def sd(n):
    s = 0
    while n:
        s += n % 6
        n //= 6
    return s

x = 1
while True:
    n = 36**17 - 6**x + 71
    if sd(n) == 61:
        print(x)
        break
    x += 1
```

Если взять большое число в первом слагаемом и перевести его в пятиричную систему, только с меньшей стеенью, то можно увидеть, что в начале будет стоять 141, а за ним нули. Значит нужно найти число x, чтобы в значении выражения $5^{55}-x$ было 54 нуля.
```python
def conv(n):
  s = ''
  while n:
    s += str(n % 5)
    n //= 5
  return s
  
for x in range(5555, 0, -1):
  n = 5**55 - x
  if conv(n).count('4') == 54:
    print(x)
    break
```
