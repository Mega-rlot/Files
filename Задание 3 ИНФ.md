# Вертикальный поиск(ВПР или VLOOKUP)
Разделяем части с помощью точки с запятой
1. В первой части ВПР выделяем ячейку, по которой хотим объединить таблицы
2. Во второй части выделяем все столбцы таблицы, которую хотим присоединить
3. В третьей части пишем номер столбца присоединённой таблицы, который хотим вывести, отсчитывая с 1
4. В четвёртой части просто пишем 0
Пример:
=ВПР(D2;$Товар.A:F;6;0)

Дальше желательно подписать новые столбцы, отфильтровать по заданию и найти ответ
