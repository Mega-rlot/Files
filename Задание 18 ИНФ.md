Чтобы найти максимальное / минимальное кол-во монет, которое может собрать робот, нужно:
1. Сделать копию таблицы
2. Рассчитаем кол-во монет в каждой ячейке первого столбца формулой: кол-во монет из ячейки сверху **+** кол-во монет из текущей ячейки из оригинальной таблицы
3. Растягиваем формулу на весь первый столбик
4. Рассчитаем кол-во монет в каждой ячейке первой строки формулой: кол-во монет из ячейки слева **+** кол-во монет из текущей ячейки из оригинальной таблицы
5. Растягиваем формулу на всю первую строку
6. Для остальных ячеек воспользуемся функцией МАКС / МИН, чтобы выбрать максимальное / минимальное значение между значениями в ячейке слева и ячейке справа, а затем к нему прибавить значение текущей ячейки из оригинальной таблицы
7. Растягиваем формулу на остальные ячейки
8. Чтобы вернуть границы, нужно скопировать оригинальную таблицу, выделяем левую верхнюю ячейку второй таблицы, нажимаем правой кнопкой мыши и в параметрах вставки выбираем пункт "оформление" (или после выделения оригинальной таблицы выделяем пункт со значком кисточка и выделяем новую таблицу)
9. Для ячеек, которые ниже границ, вставляем формулу, как для первой строки таблицы
10. Растягиваем на область ячеек под границей
11. Делаем так со всеми областями ячеек, находящимися под границей
12. Снова скопируем и вставим оформление из первой таблицы
13. Для ячеек, которые справа от границ, вставляем формулу, как для первого столбца таблицы
14. Растягиваем вниз на область ячеек правее таблицы
15. Делаем так со всеми областями ячеек, находящимися правее таблицы
16. Находим все ячейки, которые справа и снизу ограничены границами и выбираем максимальное значение из них с помощью функции МАКС и перепроверяем самостоятельно, выделив их другим цветом
17. Если кроме максимального значения в задании нужно найти ещё и минимальное, то в "найти и заменить" выбираем "формулы" вместо "значения", в найти пишем "МАКС", а в заменить "МИН"
18. Находим все ячейки, которые справа и снизу ограничены границами и выбираем минимальное значение из них с помощью функции МИН и перепроверяем самостоятельно, выделив их другим цветом
